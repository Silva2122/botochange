
1. Базовые настройки.
Файл settings.py

1.1 С верхним блоком настроек все понятно вроде должно быть.

1.2 Киви
number = +7
qiwi_token = 'qiwi_token'
Номер киви и токен, получаемый через qiwi.com/api

1.3 Яндекс
ya_number = ххх
ya_token = 'yandex_token'
Номер кошелька яндекса и токен.
!!! У яндекса БОЛЬШИЕ заморочки с получением токена. Очень большие. Проще не использовать, чем пробиться через его получение.
Если очень надо - то для помощи в этом файл 
tost.py
и еще поколдовать в файле
main.py
районе строки 1122

1.4 Телеграм
telegram_token='telegram_token'
Токен бота, очевидно. Полный.

1.5 
uah_to_rub = 2.16
usd_to_rub = 57.85
eur_to_rub = 67.73
Курсы валют, я не видел, чтобы использовались кем-то.

1.6
admins = [userid, userid, userid]
Через запятую указываются userid тех пользователей, кто будет иметь доступ к админке.

1.7
bans = []
Через запятую указываются userid тех пользователей, кто вообще не имеет доступ к боту.

1.8 Настройки хуков (это для тех, кто понимает о чем речь)
WEBHOOK_HOST = 'xx.xxx.xxx.xx'
Внешний IP сервера, куда будет стучаться телеграм. Внешний порт всегда 443.
WEBHOOK_PORT = 8000
Внутренний порт, на котором будет слушать скрипт и на который будет перенаправляться обращение снаружи
WEBHOOK_LISTEN = '127.0.0.1'
Внутренний IP, на который будет перенаправляться обращение снаружи

WEBHOOK_SSL_CERT = '/etc/nginx/keys/webhook_cert.key'
WEBHOOK_SSL_PRIV = '/etc/nginx/keys/webhook_pkey.pem'
Сертификаты, в случае использования self-signed сертификатов.
!!! При использовании таких сертификатов нужно раскомментировать в файле
main.py
одну из двух строк 1190 или 1191. Работать будет или одна или другая, со второго раза попадете.

1.9 Ошибки и ценная информация записываются в файл
workLog.txt


2. Дополнительные настройки

2.1 Если не используется qiwi, то во избежание лишних ошибок рекоммендуется в файле 
functions.py
в начале строки номер 87 убрать символ #

2.2 Если не используюеся yandex, то во избежание лишних ошибок рекоммендуется в файле 
functions.py
в начале строки номер 123 убрать символ #
